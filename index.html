How dose Javascript Work?

JS short form of the javaScript. JavaScript is called a high label Interpreted language. The javaScript allow you to make web pages more interactive by accessing and modifying the content and markup used in a web page. while it is being viewed in the browser. Also you can use javascript to add elemnets, attributes, and text to the page, or remove them . for example:

  • Add a paragraph of text after the first<>h1
  • Change the value of class atributes to trigger new CSS rulse for this elements
  • Change the size or position of an img element

How dose Javascript Event Loop work?

Event are the browser's way of indicating when something has happened(such as when a page has finished loading or a button has been clicked). Binding is the process of stating which event you are waiting to happend and which element you are waiting for that event to happend upon. When an event occurs on an element, it can triggera javaScript function. When this function then changes the web page in some way, it feels interective because it has response to the user.

LocalStorage Vs SessionStorage

The localStorage and the SessionStorage behavior are same but the different is the localStorage keeps store user information data and this data will not be delete when the user closed the browser window it will be available.
On the other hand sessionStorage data will be deleted when the user will closed the web browser by user.
